# Bike-Sales-Data(Interactive Dashboard creation using Microsoft Excel)

## Project Objective
 The objective of the project is to analyze raw customer data to identify key factors that influence bike purchases.The project aims to build pivot tables to analyze data across gender,age,region,marital status,education and commute distance and to develop an interactive Bike Sales Dashboard for business users to understand Demographic patterns in bike puchasing,Regional and lifestyle based buying behaviour.It helps stakeholders make data-driven sales and marketing decisions.
## Dataset used:
- <a href="https://github.com/Varshanatchathra/Bike-Sales-Dashboard/blob/main/Bike%20sales%20dashboard.xlsx" >Dataset</a>

## Problem statements-Chart requirements:

- Customer age bracket trends on bikes purchased.
- Average income per purchase based on gender.
- Bikes purchasd based on Commute distance.
  
- Dashboard Interaction <a href="https://github.com/Varshanatchathra/Bike-Sales-Dashboard/blob/main/Bike%20sales%20dashboard%20-%20Excel%2019-03-2026%2010_28_52%20AM.png">View Dashboard</a>

## Process

- Inserted columns to create efficient columns.
- Performed data cleaning operations like Handling missing values,Removing duplicate records and data type correction.
- Find and Replaced Marital status and Gender to understandable form.
- Inserted Pivot tables for anlayzing the bikes purchased based on commute distance,Average income per purchase,Customer age bracket.
- Created interactive dashboards in MS Excel including slicers for interactive filtering (Education,Region,Marital status),Pivot charts(line chart,clustered column charts).
